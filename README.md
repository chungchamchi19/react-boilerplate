# Introduction

This project is the boiler template for ReactJS with Vite, TailwindCSS, Jest, Eslint, Prettier.

## Setup

### Install dependency

```
yarn
```

### Start application in development mode

```
yarn dev
```

### Run test

```
yarn test --watch
```

### Run test coverage

```
yarn test --coverage
```

### Build

```
yarn build
```

### Preview application in production mode

```
yarn preview
```
